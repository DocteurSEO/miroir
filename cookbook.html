Tutoriel Interactif

Étape 1: Le Cœur de Miroir.js

miroir.js synchronise vos données JavaScript avec votre HTML. La magie opère grâce à miroir.create(), qui rend un objet "réactif".

Quand une propriété de cet objet change, le HTML est mis à jour instantanément.

  1. Afficher une donnée : Utilisez la syntaxe {{ variable }}.
  2. Lier un champ : Utilisez m-model="variable" sur un <input> pour une liaison dans les deux sens (JS ↔ HTML).

À vous de jouer : Modifiez le nom ou l'âge dans le code ou directement dans les champs de la prévisualisation.

Astuce : Essayez de changer la valeur d'âge à plus de 100 ans !

Étape 2: La Puissance des Extensions

Les extensions sont des super-pouvoirs que vous ajoutez avec miroir.extend(). Elles créent des attributs HTML personnalisés pour des logiques réutilisables.

Nous allons utiliser une extension très commune : m-show. Elle affiche un élément seulement si la variable associée est true.

Exercice : Cochez la case dans la prévisualisation pour faire apparaître et disparaître le message secret.

Astuce : Vous pouvez ajouter d'autres extensions comme m-class ou m-if pour plus de fonctionnalités.

Prévisualisation

Étape 3: Créez Votre Extension

À votre tour ! Créer une extension est un jeu d'enfant. miroir.extend() prend deux arguments :

  1. Le nom de l'attribut (ex: 'm-color').
  2. Une fonction qui reçoit l'élément (el), la valeur de l'attribut (prop), et l'état (state).

Votre mission : Créez une extension m-color qui change la couleur du texte d'un élément. Le code est presque complet, il ne manque qu'une ligne !

Astuce : Utilisez el.style.color pour changer la couleur du texte.
